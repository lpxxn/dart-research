# Flutter 导航与路由 — 系统教程

## 🚀 运行方式

```bash
# 运行导航首页
flutter run

# 运行单个章节示例
flutter run -t lib/ch01_navigation_basics.dart
flutter run -t lib/ch02_named_routes.dart
# ... 以此类推
```

## 环境要求

- Flutter 3.x+
- Dart 3.x+
