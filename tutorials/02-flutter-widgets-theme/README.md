# Flutter 自定义控件设计 & 主题美化 — 完整教程

本教程从零开始，系统讲解 Flutter 自定义控件的设计方法与主题美化技巧。内容涵盖控件组合、自绘、全局主题配置、动态换肤等核心话题，配合可运行的示例代码，帮助你快速掌握 Flutter UI 开发的精髓。

---

## 🚀 如何运行

确保已安装 Flutter SDK，然后在项目根目录执行：

```bash
flutter run
```

即可在模拟器或真机上查看示例效果。
