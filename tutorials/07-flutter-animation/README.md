# Flutter 动画与手势 — 系统教程

本项目是 Flutter 动画与手势交互的系统学习教程，涵盖隐式动画、显式动画、交错动画、Hero 动画、物理动画、手势识别、自定义转场以及 Lottie/Rive 集成。

## 📚 章节目录

| 章节 | 文件 | 主题 |
|------|------|------|
| 第 1 章 | `ch01_implicit_animations.dart` | 隐式动画 |
| 第 2 章 | `ch02_explicit_animations.dart` | 显式动画 |
| 第 3 章 | `ch03_staggered_animations.dart` | 交错动画 |
| 第 4 章 | `ch04_hero_animations.dart` | Hero 动画 |
| 第 5 章 | `ch05_physics_animations.dart` | 物理动画 |
| 第 6 章 | `ch06_gestures.dart` | 手势识别与处理 |
| 第 7 章 | `ch07_custom_transitions.dart` | 自定义页面转场 |
| 第 8 章 | `ch08_lottie_rive.dart` | Lottie 与 Rive 动画 |
| 第 9 章 | `ch09_animation_practice.dart` | 动画综合实战 |

## 🚀 运行方式

### 运行导航首页

```bash
flutter run
```

### 运行单个章节示例

```bash
flutter run -t lib/ch01_implicit_animations.dart
flutter run -t lib/ch02_explicit_animations.dart
flutter run -t lib/ch03_staggered_animations.dart
flutter run -t lib/ch04_hero_animations.dart
flutter run -t lib/ch05_physics_animations.dart
flutter run -t lib/ch06_gestures.dart
flutter run -t lib/ch07_custom_transitions.dart
flutter run -t lib/ch08_lottie_rive.dart
flutter run -t lib/ch09_animation_practice.dart
```
