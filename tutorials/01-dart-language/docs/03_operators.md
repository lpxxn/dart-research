# 第 3 章：运算符

Dart 提供了丰富的运算符，涵盖算术、逻辑、位运算、赋值、条件、级联、展开和类型测试等。本章将按类别逐一讲解，每个运算符配有代码示例。

## 3.1 算术运算符

基本的数学运算操作：

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `+` | 加 | `2 + 3` → `5` |
| `-` | 减 | `5 - 2` → `3` |
| `*` | 乘 | `3 * 4` → `12` |
| `/` | 除（返回 double） | `7 / 2` → `3.5` |
| `~/` | 整除（返回 int） | `7 ~/ 2` → `3` |
| `%` | 取余 | `7 % 3` → `1` |

**注意 `/` 和 `~/` 的区别**：在 Dart 中 `/` 运算符始终返回 `double`，即使两个操作数都是整数。如果需要整数除法结果，必须使用 `~/`：

```dart
print(7 / 2);    // 3.5（double）
print(7 ~/ 2);   // 3（int）
```

## 3.2 自增自减运算符

| 运算符 | 含义 | 说明 |
|--------|------|------|
| `++x` | 前缀自增 | 先加 1，再返回值 |
| `x++` | 后缀自增 | 先返回值，再加 1 |
| `--x` | 前缀自减 | 先减 1，再返回值 |
| `x--` | 后缀自减 | 先返回值，再减 1 |

前缀和后缀的区别在于返回值的时机：

```dart
var a = 5;
print(++a);   // 6（先加后返回）
print(a++);   // 6（先返回后加，此时 a 已经是 7）
print(a);     // 7
```

## 3.3 比较运算符

| 运算符 | 含义 |
|--------|------|
| `==` | 相等 |
| `!=` | 不等 |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |

**关于 `==`**：Dart 的 `==` 调用的是对象的 `operator==` 方法，而不是简单的引用比较。对于基本类型和 `String`，`==` 比较的是值是否相等：

```dart
var s1 = 'Hello';
var s2 = 'Hello';
print(s1 == s2);                // true（值相等）
print(identical(s1, s2));       // true 或 false（引用是否相同，取决于实现）
```

你可以在自定义类中重写 `operator==` 和 `hashCode` 来定义相等的语义。

## 3.4 逻辑运算符

| 运算符 | 含义 | 说明 |
|--------|------|------|
| `&&` | 逻辑与 | 两者为 true 才为 true |
| `\|\|` | 逻辑或 | 任一为 true 即为 true |
| `!` | 逻辑非 | 取反 |

**短路求值**：逻辑运算符使用短路求值策略：
- `&&`：如果左侧为 `false`，右侧不会被执行
- `||`：如果左侧为 `true`，右侧不会被执行

```dart
bool check() {
  print('check() 被调用');
  return true;
}

false && check();  // check() 不会被调用
true || check();   // check() 不会被调用
```

## 3.5 位运算符

位运算直接操作整数的二进制位，在性能敏感和底层编程中非常有用：

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `&` | 按位与 | `0xF0 & 0x0F` → `0x00` |
| `\|` | 按位或 | `0xF0 \| 0x0F` → `0xFF` |
| `^` | 按位异或 | `0xFF ^ 0x0F` → `0xF0` |
| `~` | 按位取反 | `~0x0F` → ... |
| `<<` | 左移 | `1 << 3` → `8` |
| `>>` | 右移 | `8 >> 2` → `2` |

### 应用场景：权限标志位

位运算在权限管理中非常常见。每个权限占一个比特位，通过位运算组合和检查权限：

```dart
const read    = 1 << 0;  // 0001 = 1
const write   = 1 << 1;  // 0010 = 2
const execute = 1 << 2;  // 0100 = 4

// 组合权限：使用按位或
var permission = read | write;  // 0011 = 3

// 检查权限：使用按位与
bool canRead = (permission & read) != 0;     // true
bool canExec = (permission & execute) != 0;  // false

// 添加权限
permission |= execute;  // 0111 = 7

// 移除权限
permission &= ~write;   // 0101 = 5
```

## 3.6 赋值运算符

Dart 提供了基础赋值和多种复合赋值运算符：

| 运算符 | 等价于 | 说明 |
|--------|--------|------|
| `=` | — | 基本赋值 |
| `??=` | `a = a ?? b` | 仅当 a 为 null 时赋值 |
| `+=` | `a = a + b` | 加后赋值 |
| `-=` | `a = a - b` | 减后赋值 |
| `*=` | `a = a * b` | 乘后赋值 |
| `/=` | `a = a / b` | 除后赋值 |
| `~/=` | `a = a ~/ b` | 整除后赋值 |
| `%=` | `a = a % b` | 取余后赋值 |
| `&=` | `a = a & b` | 按位与后赋值 |
| `\|=` | `a = a \| b` | 按位或后赋值 |
| `^=` | `a = a ^ b` | 按位异或后赋值 |
| `<<=` | `a = a << b` | 左移后赋值 |
| `>>=` | `a = a >> b` | 右移后赋值 |

**`??=`（空值赋值）** 是 Dart 特有的运算符，仅在变量为 `null` 时才执行赋值：

```dart
String? name;
name ??= 'Dart';   // name 为 null，所以赋值 'Dart'
print(name);        // 'Dart'
name ??= 'Flutter'; // name 不为 null，不执行赋值
print(name);        // 仍然是 'Dart'
```

## 3.7 条件运算符

### 三目运算符 `? :`

```dart
var score = 85;
var grade = score >= 60 ? '及格' : '不及格';
print(grade);  // '及格'
```

### 空合并运算符 `??`

如果左侧表达式不为 `null` 则返回左侧值，否则返回右侧值：

```dart
String? nickname;
var displayName = nickname ?? '匿名用户';
print(displayName);  // '匿名用户'

nickname = 'Dart 爱好者';
displayName = nickname ?? '匿名用户';
print(displayName);  // 'Dart 爱好者'
```

## 3.8 级联运算符 `..` 和 `?..`

级联运算符允许对同一个对象连续调用多个方法或设置多个属性，无需重复引用对象名。级联返回的是对象本身，而不是方法的返回值：

```dart
var sb = StringBuffer()
  ..write('Hello')
  ..write(', ')
  ..write('Dart!')
  ..writeln();
print(sb);  // 'Hello, Dart!\n'
```

等价于：

```dart
var sb = StringBuffer();
sb.write('Hello');
sb.write(', ');
sb.write('Dart!');
sb.writeln();
```

**`?..`（空安全级联）**：当对象可能为 null 时使用，如果对象为 null 则整个级联链不执行：

```dart
List<int>? list;
list?..add(1)..add(2)..add(3);  // list 为 null，不执行任何操作
```

## 3.9 展开运算符 `...` 和 `...?`

展开运算符用于在集合字面量中将另一个集合的所有元素"展开"插入：

```dart
var list1 = [1, 2, 3];
var list2 = [0, ...list1, 4, 5];
print(list2);  // [0, 1, 2, 3, 4, 5]
```

**`...?`（空安全展开）**：如果集合可能为 null，使用 `...?` 避免空指针异常：

```dart
List<int>? nullableList;
var result = [0, ...?nullableList, 4];
print(result);  // [0, 4]

nullableList = [1, 2, 3];
result = [0, ...?nullableList, 4];
print(result);  // [0, 1, 2, 3, 4]
```

展开运算符同样适用于 `Set` 和 `Map`：

```dart
var set1 = {1, 2, 3};
var set2 = {0, ...set1, 4};     // {0, 1, 2, 3, 4}

var map1 = {'a': 1, 'b': 2};
var map2 = {'z': 0, ...map1};   // {'z': 0, 'a': 1, 'b': 2}
```

## 3.10 类型测试运算符

| 运算符 | 含义 |
|--------|------|
| `is` | 如果对象是指定类型则返回 true |
| `is!` | 如果对象不是指定类型则返回 true |
| `as` | 强制类型转换（失败抛异常） |

```dart
Object value = 'Hello';
if (value is String) {
  // 类型提升：此处 value 被视为 String
  print(value.toUpperCase());
}

String s = value as String;  // 强制转换
```

详细内容参见第 2 章 2.5 节。

## 3.11 运算符优先级

从高到低排列（同一行优先级相同）：

| 优先级 | 运算符 |
|--------|--------|
| 16 | `()` 函数调用、`[]` 下标、`.` 成员访问、`?.` |
| 15 | `!expr`、`-expr`（一元）、`~expr`、`++`、`--` |
| 14 | `*`、`/`、`~/`、`%` |
| 13 | `+`、`-` |
| 12 | `<<`、`>>`、`>>>` |
| 11 | `&` |
| 10 | `^` |
| 9 | `\|` |
| 8 | `>=`、`>`、`<=`、`<`、`as`、`is`、`is!` |
| 7 | `==`、`!=` |
| 6 | `&&` |
| 5 | `\|\|` |
| 4 | `??` |
| 3 | `? :` |
| 2 | `..`、`?..` |
| 1 | `=`、`*=`、`/=` 等赋值运算符 |

**建议**：当优先级不确定时，使用括号明确运算顺序，提高代码可读性。
