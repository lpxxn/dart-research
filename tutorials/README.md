## 🚀 快速开始

```bash
# 运行某个教程项目
cd 01-dart-language
dart run bin/ch01_hello_dart.dart

# Flutter 项目运行某一章
cd 03-flutter-layout
flutter run -t lib/ch01_layout_principle.dart

# 运行测试（08-架构项目）
cd 08-flutter-architecture
flutter test
```

每个项目都有独立的 `README.md`，包含详细的章节目录和运行说明。
